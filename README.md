## prashjha.github.io
This is my personal website. This a single page website which supports multiple tabs. Code is easy to extend and modify. It includes multiple features such as 

* tabs
* social media buttons
* timeline
* gallery browser


## Logo for Research Gate
Since the website relies on fontawesome 4.7.0 which does not have icon for Research Gate, I had to modify the svg file to make it work. You can find the logo in `images/icons/researchgate.svg`.

## Template for CV
I have worked hard in desgining my own template for CV in MS Word. I have been motivated by few templates in the internet. I have designed the template from scratch instead of building on top of some free available templates. You are free to check it out and use. It is located in `files/cv.docx`.

## Reference to external sources
The website design relies on freely available contents online. I am listing sources for features below:

* Photo gallery browser: `simplelightbox` library . [Link](https://github.com/andreknieriem/simplelightbox)

* Timeline design `CSS Timeline` by Nils Wittler. See [source code](https://codepen.io/NilsWe/pen/FemfK/).

* Social media buttons `SMIcons2` by Helen. See [source code](https://codepen.io/CF-251/pen/bovZrv) and [list](https://www.templatemonster.com/blog/how-to-add-social-media-icons-website/) of some other interesting designs.

* Hoverable dropdown buttons: [source](https://www.w3schools.com/howto/howto_css_dropdown.asp)
