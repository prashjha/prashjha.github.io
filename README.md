## prashjha.github.io
This is a repository for my [personal website](https://prashjha.github.io/). It is a single html file website. Website is organized using tabs. Currently, it has followsing features

* tabs
* social media buttons
* timeline
* gallery browser

## Template for CV
With lot of hard work, I have designed a new template for my CV. I have built this from scratch instead of building it on top of freely available templates. You are free to check it out and use. It is located in `files/cv.docx` [link](files/cv.docx).

## Reference to external sources
Website relies on freely available programming scripts online. I am listing sources for features below:

* Photo gallery browser: `simplelightbox` library. [Link](https://github.com/andreknieriem/simplelightbox)

* Timeline design `CSS Timeline` by Nils Wittler. [Source code](https://codepen.io/NilsWe/pen/FemfK/)

* Social media buttons `SMIcons2` by Helen. See [source code](https://codepen.io/CF-251/pen/bovZrv) and [list](https://www.templatemonster.com/blog/how-to-add-social-media-icons-website/) of some other interesting designs.

* Hoverable dropdown buttons: [source](https://www.w3schools.com/howto/howto_css_dropdown.asp)

* Academicons from James Walsh: [Github](https://github.com/jpswalsh/academicons)
